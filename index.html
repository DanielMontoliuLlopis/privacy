MIRROR ALARM

Política de Privacidad

Última actualización: 06/12/2025
Procesamiento 100% en tu dispositivo · Sin subir fotos a servidores

Resumen rápido: Esta aplicación usa la cámara solo para verificar que estás despierto mediante una foto. Las imágenes se procesan exclusivamente en tu dispositivo, no se almacenan permanentemente, no se envían a servidores y no se realiza reconocimiento facial, únicamente detección de presencia de una cara.

2. Datos que Recopilamos

La aplicación no recopila datos personales identificables que se envíen a servidores externos. Todo el tratamiento se realiza de forma local en el dispositivo del usuario.

2.1 Cámara

La aplicación solicita acceso a la cámara con la única finalidad de:

  • Permitir al usuario tomar una fotografía para poder apagar la alarma.

Las fotografías tomadas:

  • Se procesan únicamente en el dispositivo del usuario (on-device).
  • No se envían a ningún servidor externo.
  • No se almacenan de forma permanente por la aplicación.
  • No se usan para reconocimiento facial, únicamente para detectar la presencia de una cara.

2.2 Notificaciones

La aplicación utiliza notificaciones locales para:

  • Activar y mostrar alarmas en la hora configurada.
  • Informar al usuario de que una alarma está sonando y debe ser atendida.

2.3 Almacenamiento Local (en el dispositivo)

La aplicación almacena en el propio dispositivo únicamente:

  • Configuración de alarmas (hora, días, estado activado/desactivado).
  • Preferencias de la aplicación (por ejemplo, volumen o tono).

Estos datos se almacenan únicamente para el correcto funcionamiento de la aplicación y no se envían a servidores externos.

3. Tratamiento de Imágenes (Detección de Persona)

Para apagar la alarma, la aplicación puede requerir que el usuario tome una fotografía desde la propia app.

En relación con esta funcionalidad:

  • La fotografía se procesa exclusivamente en el dispositivo para detectar la presencia de una cara.
  • La imagen nunca se envía a servidores ni a terceras partes.
  • No se realiza reconocimiento facial (no se identifica quién es la persona), solo se detecta si hay una cara en la imagen.
  • Tras el procesamiento, la imagen puede ser eliminada de forma automática o permanecer como archivo temporal del sistema, sin que la aplicación realice un almacenamiento permanente ni un uso adicional.

4. Qué NO hacemos con tus datos

La aplicación NO realiza ninguna de las siguientes acciones:

  • No comparte datos con terceros.
  • No vende datos a anunciantes u otras empresas.
  • No utiliza las fotografías para entrenar modelos externos.
  • No ejecuta reconocimiento facial para identificar personas.
  • No utiliza servidores externos para guardar tus fotos.
  • No realiza publicidad personalizada basada en tus datos.

5. Seguridad de la Información

Se adoptan medidas razonables para garantizar que la información que se procesa (principalmente la configuración de alarmas y las imágenes temporales) se maneja de forma segura y solo en el dispositivo.

El acceso a la cámara únicamente se realiza cuando el usuario concede los permisos correspondientes y en el momento en que es necesario para el uso de la funcionalidad principal (apagar la alarma con foto).

6. Bases Legales del Tratamiento

El tratamiento de los datos estrictamente necesarios para el funcionamiento de la aplicación se fundamenta en:

  • Consentimiento del usuario: al otorgar permisos de cámara y notificaciones.
  • Interés legítimo: garantizar el funcionamiento correcto de la alarma y de las funcionalidades asociadas a ella.

7. Terceros y Bibliotecas Utilizadas

La aplicación puede hacer uso de bibliotecas de terceros para:

  • Manejo de la cámara.
  • Gestión de notificaciones y alarmas locales.
  • Detección de rostro en el propio dispositivo (on-device).

Estas bibliotecas pueden ser, por ejemplo, servicios como Google ML Kit para detección de rostro en el dispositivo. En cualquier caso:

  • El procesamiento se realiza localmente en el dispositivo.
  • No se envían las imágenes a servidores externos desde la aplicación.

8. Uso por Menores de Edad

La aplicación está destinada a usuarios mayores de 13 años. No se recopila de forma intencionada información de menores de edad.

Si un padre, madre o tutor legal considera que un menor ha proporcionado datos, puede contactar con el responsable para solicitar información o la eliminación de dichos datos almacenados localmente (en la medida en que sea posible).

9. Derechos del Usuario

Aunque la aplicación no almacena datos personales en servidores del responsable, el usuario tiene derecho a:

  • Obtener información clara sobre cómo se tratan sus datos.
  • Solicitar la eliminación de la configuración almacenada localmente (por ejemplo, eliminando la aplicación o borrando los datos desde los ajustes del dispositivo).

Para cualquier consulta o solicitud relacionada con sus derechos, el usuario puede contactar al correo: llodamont@gmail.com.

10. Cambios en esta Política de Privacidad

Esta Política de Privacidad puede ser actualizada para reflejar cambios en la aplicación o en la normativa aplicable.

En caso de cambios relevantes, se informará al usuario mediante una actualización de la aplicación y/o un aviso dentro de la propia app.
